# Sheet Template

Gunakan `CommTECH_Insight_2026_Sheet_Template.xlsx` sebagai starter Google Sheet untuk app ini.

Langkah cepat:

1. Upload `.xlsx` ke Google Drive.
2. Buka file sebagai Google Sheets.
3. Isi tab `02 RD 2026`, `Prep`, dan `List Peserta`.
4. Share sheet ke Google service account.
5. Copy spreadsheet ID ke `GOOGLE_SHEET_ID`.

Jangan geser urutan kolom A-K di `02 RD 2026`, karena app membaca posisi kolom tersebut.
